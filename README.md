# 🎰 Python Slot Machine Game

A clean and interactive terminal-based Slot Machine Game built with Python.

This project simulates a simple slot machine where the player starts with a balance, places bets, spins random symbols, and earns payouts based on matching combinations.

It is designed to demonstrate Python fundamentals, game logic, input validation, function-based structure, and beginner-to-intermediate programming skills.

---

# 📌 Project Overview

The Python Slot Machine Game allows the player to:

* Start with a fixed balance
* Place a valid bet
* Spin a row of random slot symbols
* Win rewards if all three symbols match
* Continue playing until they quit or run out of money
* View a final game summary

This project focuses on writing clean, readable, and organized Python code while building a complete playable terminal game.
